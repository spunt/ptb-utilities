Psychtoolbox Utilities

About

A collection of utility functions for performing some common actions using the stimulus presentation software Psychophysics Toolbox Version 3 (PTB-3). For more information on PTB-3, see their wiki. Some of these functions were inspired by functions written by Dr. Emily Falk, who is awesome and has an awesome lab website.
